# Air_quq<br>
## 개요
* 현재 서비스 중인 Air_bnb를 벤치마킹하여 소유 중인 집이나 기술을 공유하는 Air_quq입니다.
* 인원 : 5명
* 개발 기간 : 2020.07.01 ~ 2020.08.03
## 맡은 기능
* 메인 UI, 온라인 체험(웹캠), 예약 현황 목록, 등록한 체험 관리, 평점 및 리뷰 등록, 숙소 및 체험 신고 등록

## 사용 기술
* Maria DB 10.3.21
* HTML, CSS, JavaScript(jQuery, Ajax), BootStrap
* Java, Spring, Node.js
